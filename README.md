# ⚡ AI vs Human Platform

An interactive web platform that compares Artificial Intelligence and Human intelligence through engaging games, quizzes, and visual simulations.

---

## 🚀 Features

- ⚔ Battle Arena (Quiz System)
- 🎮 Guess AI vs Human Game
- 🌌 Animated Background & Particles
- ✨ Modern UI (Glassmorphism + Glow Effects)
- 🎯 Typing Animation & Hover Effects
- 📊 Score Tracking System

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations + Effects)
- JavaScript (Game Logic)
- Particles.js
- Vanilla Tilt.js
